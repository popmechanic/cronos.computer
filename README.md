# cronos.computer

A birth chart calculator and astrology app. Enter your birth date, time, and location to generate your natal chart with planetary positions across the zodiac.

## Features

- Calculate planetary positions for any birth date/time/location
- Display all major planets and points (Sun, Moon, Mercury, Venus, Mars, Jupiter, Saturn, Uranus, Neptune, Pluto, North Node, Chiron, Lilith)
- Show zodiac sign placements with degrees
- Element and modality breakdowns
- Built-in geocoding for major world cities

## Tech Stack

- React (via Babel standalone)
- Tailwind CSS
- Fireproof (local-first database)

## Usage

Open `index.html` in a browser or serve via any static file server.
